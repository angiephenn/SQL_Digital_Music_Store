# Description:
This project aims to analyze a digital music store database using SQLite and Excel to gain insights into the store's media, customers, and sales. The objective includes:

- Determine the most played genres from 2009-2013 based on playlist data
- Identify the top 10 best-selling genres (2009-2013) by total amount paid per genre
- Explore the demand for rock music in different countries
- Analyze the monthly customer payments for rock tracks (2009-2013) to understand payment trends
